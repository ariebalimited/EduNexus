# 🎓 EduNexus
**Empowering students with AI-driven learning and seamless collaboration.**

EduNexus is an educational platform that integrates GPT-4o to provide students with a high-performance study environment, featuring advanced analytics and unlimited resources.

## 🚀 Key Features
*   **🤖 AI-Powered Learning:** Full access to GPT-4o for unlimited daily queries and custom study plans.
*   **📝 Unlimited Exams & Analytics:** Practice with up to 500 exams and track progress with advanced performance reports.
*   **👥 Collaboration Hub:** Unlimited study groups and built-in tools for student teamwork.
*   **☁️ Massive Storage:** 100GB of dedicated cloud storage for all your study materials and video tutorials.

## 🛠️ Tech Stack
*   **Framework:** [Streamlit](https://streamlit.io) (Frontend/Backend)
*   **Database:** [Supabase](https://supabase.com) (Auth & Data Storage)
*   **AI Engine:** [OpenAI GPT-4o](https://openai.com)
*   **Language:** Python 3.10+
